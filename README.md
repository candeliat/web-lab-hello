## Projeyi Yerelde Çalıştırma
Projeyi kendi bilgisayarınızda çalıştırmak için şu komutları izleyin:

1. Depoyu klonlayın: `git clone [GITHUB_REPO_LINKINIZ]`
2. Klasöre girin: `cd web-lab-hello`
3. Bağımlılıkları kurun: `npm install`
4. Projeyi başlatın: `npm run dev`
